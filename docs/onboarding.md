# ForgeFlow Onboarding

Welcome to ForgeFlow. Here is the fastest way to get productive:

1) Configure project scan roots
   - Run **ForgeFlow: Configure Project Scan Roots**
   - Add your top-level repo folders (multi-root supported)

2) Open the ForgeFlow views
   - Activity Bar: **ForgeFlow** icon
   - Panel: **ForgeFlow: Dashboard**

3) Verify entry points
   - Open a project and check **Build Scripts** and **Entry Points**
   - Right-click any file to **Add Entry Point** if needed

4) Configure PowerShell profiles (optional)
   - Settings: `forgeflow.powershell.profiles`
   - Set default: `forgeflow.powershell.defaultProfileId`

5) Dashboard tokens (optional)
   - Run **ForgeFlow: Configure Dashboard Tokens** for GitHub/GitLab/Azure

Need to re-run onboarding? Use **ForgeFlow: Start Onboarding** anytime.
