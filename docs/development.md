# Development

## Requirements
- Node.js 20+
- VS Code 1.90+

## Install
```bash
npm install
```

## Build
```bash
npm run compile
```

## Typecheck
```bash
npm run typecheck
```

## Test
```bash
npm test
```

## Run in VS Code
- Press `F5` to launch the Extension Development Host.

## Package
```bash
npm run package
```
