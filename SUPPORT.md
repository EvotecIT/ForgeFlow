# Support

ForgeFlow support is handled through GitHub:

- Bug reports: https://github.com/EvotecIT/ForgeFlow/issues
- Feature requests: https://github.com/EvotecIT/ForgeFlow/issues
- Source code: https://github.com/EvotecIT/ForgeFlow

When reporting an issue, include the VS Code version, ForgeFlow version, operating system, and the relevant command or view you were using.
